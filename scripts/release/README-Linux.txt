Neurovox Stroke Eye Monitor — Linux
------------------------------------
1. Extract the archive (name matches the .tar.gz you downloaded).
2. Run from a terminal:
     cd NeurovoxStrokeEyeMonitor-Linux
     ./NeurovoxStrokeEyeMonitor

If the binary is not executable:  chmod +x NeurovoxStrokeEyeMonitor

You may need OpenGL/GLib libraries on minimal distros (e.g. libgl1, libglib2.0-0 on Debian/Ubuntu).

Place a .env file with API keys in the same directory as the binary, or in your working directory, as described in the project README.
